## セットアップ手順

### 1. pnpm 有効化

```bash
corepack enable
corepack prepare pnpm@10.9.0 --activate
```

pnpm がグローバルにインストールされていなかったため、corepack で有効化。

### 2. Git submodule 初期化

```bash
git submodule update --init --recursive
```

`packages/better-call`（BetterAuth 統合）がサブモジュール。初期化しないと `pnpm install` が失敗する。

### 3. 依存関係インストール

```bash
pnpm install
```

循環依存の警告あり（core ↔ tools ↔ bun-websockets、sdk ↔ shared-types）。動作には影響なし。

### 4. ビルド

```bash
# メインビルド（esbuild — CJS/ESM + .d.ts）
npx tsx build.ts

# SDK（Rollup）
cd packages/sdk && pnpm build

# Monitor（Vite）
cd packages/monitor && pnpm build

# Playground（Vite）
cd packages/playground && pnpm build
```

**注意:** `pnpm build` は `tsx` がグローバルにないと失敗するため `npx tsx build.ts` を使用。`pnpm build-all` は better-call の `tsdown` がないため失敗。個別ビルドで回避。

### 5. Redis 起動

```bash
brew install redis
brew services start redis
redis-cli ping  # → PONG
```
