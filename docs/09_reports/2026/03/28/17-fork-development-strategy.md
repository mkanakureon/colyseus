## なぜフォーク（ソースコード同梱）か

### npm install だけでも動く

```bash
mkdir mmo && cd mmo
npm init -y
npm install colyseus @colyseus/ws-transport
```

これだけで Room を作って動かせる。**機能的には十分。**
