### WorldRoom — ゾーン管理

```typescript
  maxClients = 50;  // 1ゾーン最大50人

  onCreate(options) {
    this.setState(new WorldState());
    // ゾーンのNPC・オブジェクトをDBからロード
    this.loadZone(options.zoneId);

    // NPCのAIループ（1秒ごと）
    this.setSimulationInterval(() => this.updateNPCs(), 1000);
  }

  onAuth(client, options, context) {
    // kaedevn の JWT トークンで認証（既存auth連携）
    return verifyKaedevnToken(context.token);
  }

  onJoin(client, options, auth) {
    // プレイヤーをゾーンに追加
    const player = new Player();
    player.name = auth.username;
    player.avatar = auth.avatarPreset; // 図形キャラプリセット
    this.state.players.set(client.sessionId, player);
  }

  onMessage(client, "move", { direction }) {
    // 隣接ゾーンへ移動 → 別のWorldRoomへ遷移
    const nextZone = this.getAdjacentZone(direction);
    if (nextZone) {
      client.send("zone_change", { zoneId: nextZone });
      // クライアント側で leave → 新しい WorldRoom に join
    }
  }

  onMessage(client, "interact", { targetId }) {
    // NPC/オブジェクトへのインタラクション
    // → StoryRoom を生成してイベント開始
  }

  onDispose() {
```

```typescript
class StoryRoom extends Room<StoryState> {
  maxClients = 4;  // 1イベント最大4人（協力プレイ）

  onCreate(options) {
    this.setState(new StoryState());
    this.npcId = options.npcId;
    this.context = options.storyContext;
  }

  onMessage(client, "say", { text }) {
    // プレイヤーの発言 → Claude API でNPC応答生成
    const response = await generateNPCDialogue({
      npc: this.npcId,
      playerInput: text,
      context: this.context,
      otherPlayers: this.state.players,
    });

    // インラインタグ付きテキストとして配信
    // 例: "[e:smile]ようこそ旅人よ。[p:wave]この先に[e:serious]危険な森がある。"
    this.broadcast("npc_dialogue", {
      speaker: response.npcName,
      text: response.text,  // インラインディレクティブ含む
    });

    // ストーリー選択肢を提示
    if (response.choices) {
      this.broadcast("choices", response.choices);
    }
  }
}
```

### ChatRoom — チャット

```typescript
class ChatRoom extends Room {
  maxClients = 100;

  onMessage(client, "chat", { text, channel }) {
    // channel: "global" | "zone" | "party" | "whisper:{targetId}"
    switch (channel) {
      case "global":
        this.broadcast("chat", { sender: client.auth.name, text });
        break;
      case "whisper":
        // 特定クライアントにのみ送信
        const target = this.clients.find(c => c.auth.id === targetId);
        target?.send("chat", { sender: client.auth.name, text, whisper: true });
        break;
    }
  }
}
```

## AI 統合設計

### NPC 対話フロー

```
プレイヤー「この村に何かあったの？」
  ↓ WebSocket (StoryRoom)
Colyseus Server
  ↓ Claude API リクエスト
{
  system: "あなたは村の長老「ヨハン」です。村は最近モンスターに...",
  messages: [
    { role: "user", content: "この村に何かあったの？" }
  ]
}
  ↓ Claude API レスポンス
"[e:sad]ああ、旅人よ...[wait:300]この村はな、[e:serious]3日前から
魔物に襲われておるのじゃ。[cam:shake][e:angry]奴らは毎晩やってくる！
[click][e:sad][p:tired]もう、わしらの力だけでは..."
  ↓ broadcast to StoryRoom clients
全プレイヤーのテキストウィンドウに表示（インラインタグで表情・演出が自動適用）
```

### AIの特徴

- **インラインタグ付きで生成** — Claude に `[e:smile]` 等のタグ仕様を system prompt で教える
- **プレイヤー全員の入力を考慮** — 協力プレイ時、4人の発言を context に含める
- **世界状態を反映** — ゾーンの状態、クエスト進行、他プレイヤーの行動を prompt に含める
- **kaedevn のスムーズ遷移がそのまま動く** — AIが `[e:angry]` を出力すれば、全プレイヤーの画面でNPCが怒り顔にスムーズ遷移
