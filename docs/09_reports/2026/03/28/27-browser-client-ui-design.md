# テキスト型MMO ブラウザクライアント UI 設計

> **方向:** 横（landscape）
> **テキストのみ:** 画像・アイコン・絵文字なし
> **対応:** スマホ横持ち + PC ブラウザ
> **操作:** タップ / クリック / キーボード

---

## レイアウト基本構造

```
┌─────────────────────────────────────────────────┐
│ HEADER  [プレイヤー情報]          [ゾーン名]      │ 40px固定
├────────────────────────┬────────────────────────┤
│                        │                        │
│   LEFT PANEL           │   RIGHT PANEL          │
│   (メインテキスト)       │   (サブ情報/操作)       │
│                        │                        │
│   スクロール可能         │   画面ごとに変化         │
│                        │                        │
│                        │                        │
├────────────────────────┴────────────────────────┤
│ FOOTER  [選択肢 / 入力欄]                         │ 可変高さ
└─────────────────────────────────────────────────┘
```

**比率:**
- PC: 左 60% / 右 40%
- スマホ横: 左 55% / 右 45%
- 切り替え: 右パネルが不要な画面では左100%

---

## レスポンシブ設計

| 項目 | PC (1024px+) | スマホ横 (568px+) |
|------|-------------|------------------|
| フォント | 15px | 13px |
| 行間 | 1.7 | 1.5 |
| パネル分割 | 60:40 | 55:45 |
| ヘッダー | 1行 | 1行（省略表示） |
| 選択肢 | 横並び可 | 縦並び |
| 入力欄高さ | 36px | 32px |
| タッチターゲット | - | 44px最小 |

## 操作方式

### PC

| 操作 | 方法 |
|------|------|
| 選択肢 | 数字キー (1-9, 0) |
| チャット入力 | テキスト入力 → Enter |
| 次のページ | Space / Enter |
| 戻る | Escape / 0 |
| ショートカット | S:ステータス I:インベントリ C:チャット |

### スマホ横持ち

| 操作 | 方法 |
|------|------|
| 選択肢 | ボタンタップ（44px以上） |
| チャット | ソフトキーボード |
| 次のページ | 画面タップ |
| 戻る | [戻る] ボタン |
