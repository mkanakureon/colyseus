## 4. 選び方ガイド

### 4.1 Transport の選び方

| Transport | ランタイム | パフォーマンス | 互換性 | 推奨用途 |
|-----------|----------|--------------|--------|---------|
| **`@colyseus/ws-transport`** | Node.js | 標準 | 最高（デフォルト） | **一般的な用途。迷ったらこれ** |
| **`@colyseus/uwebsockets-transport`** | Node.js | **最高** | C++ バインディングのためビルド環境が必要 | 高負荷プロダクション環境 |
| **`@colyseus/bun-websockets`** | Bun | 高 | Bun ランタイム限定 | Bun で開発する場合 |
| `@colyseus/h3-transport` | Node.js | 実験的 | WebTransport 対応ブラウザ限定 | 低レイテンシが必要な実験的プロジェクト |
| `@colyseus/tcp-transport` | Node.js | - | private (非公開) | 内部用途 / 特殊ケース |

**判断フロー:**
1. Bun を使うか？ → `@colyseus/bun-websockets`
2. パフォーマンス最優先か？ → `@colyseus/uwebsockets-transport`
3. それ以外 → **`@colyseus/ws-transport`**（デフォルト）

### 4.2 Driver の選び方

| Driver | バックエンド | 用途 |
|--------|-----------|------|
| **インメモリ（デフォルト）** | なし（`@colyseus/core` 内蔵） | **開発環境。単一プロセス** |
| **`@colyseus/redis-driver`** | Redis | **プロダクション推奨。複数プロセス/サーバー対応** |
| `@colyseus/mongoose-driver` | MongoDB | MongoDB を既に使っている場合 |
| `@colyseus/drizzle-driver` | PostgreSQL 等 | RDB でマッチメイキングデータを管理したい場合 |

**判断フロー:**
1. 開発環境 / 単一プロセスか？ → デフォルト（設定不要）
2. 複数プロセス / 本番か？ → **`@colyseus/redis-driver`**
3. 既存 MongoDB があるか？ → `@colyseus/mongoose-driver`
4. RDB (PostgreSQL) を使いたいか？ → `@colyseus/drizzle-driver`

### 4.3 Presence の選び方

| Presence | バックエンド | 用途 |
|----------|-----------|------|
| **インメモリ（デフォルト）** | なし（`@colyseus/core` 内蔵） | **開発環境。単一プロセス** |
| **`@colyseus/redis-presence`** | Redis (Pub/Sub) | **複数プロセス/サーバー間通信。本番環境必須** |

**判断フロー:**
1. 単一プロセスか？ → デフォルト
2. 複数プロセス / 本番か？ → **`@colyseus/redis-presence`**

> **注意:** 複数プロセス構成では、Driver と Presence の両方を Redis にするのが一般的。`@colyseus/redis-driver` + `@colyseus/redis-presence` のセットで使う。

### 4.4 Serializer の選び方

| Serializer | 状態 | 用途 |
|------------|------|------|
| **`@colyseus/schema` (外部)** | **現行・推奨** | v4.x。全新規プロジェクトはこれを使う |
| `@colyseus/fossil-delta-serializer` | レガシー | 旧バージョンからの移行用 |
| `@colyseus/legacy-schema-serializer` | レガシー (private) | 旧 Schema API からの移行用 |

**結論:** 新規プロジェクトは `@colyseus/schema` ^4.0.7 を使う。レガシーシリアライザーは移行期間用。
