# Dev Server Skill

開発サーバー・サンプルアプリを起動する。

## トリガー

"dev-server", "サーバー起動", "example起動", "サンプル起動"

## コマンド

### Example アプリ起動

```bash
pnpm example
```

### Monitor（開発モード）

```bash
cd packages/monitor && pnpm dev
```

### Playground（開発モード）

```bash
cd packages/playground && pnpm dev
```

### ウォッチモード（パッケージ自動リビルド）

```bash
pnpm watch
```
